# Getting Started

Добро пожаловать в проект "AutoOffer"! Этот раздел поможет вам быстро начать работу с проектом.

## Что вы найдете в этом разделе

- **[Установка и настройка](installation.md)** — детальная инструкция по установке всех компонентов системы
- **[Первые шаги](first-steps.md)** — запуск проекта и базовые операции
- **[Настройка среды разработки](development-setup.md)** — оптимизация рабочего окружения

## Быстрый старт

Если вы хотите быстро запустить проект:

1. Установите зависимости (см. [Установка](installation.md))
2. Настройте переменные окружения
3. Запустите базу данных через Docker
4. Примените миграции
5. Запустите backend и frontend

Подробные инструкции смотрите в [Первые шаги](first-steps.md).

## Требования

Перед началом работы убедитесь, что у вас установлены:

- **Python 3.11+**
- **Node.js 20+**
- **PostgreSQL 16+** (или Docker для запуска БД)
- **Docker и Docker Compose** (рекомендуется)
- **Git**

## Следующие шаги
