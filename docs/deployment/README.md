# Деплой

Руководство по развертыванию проекта "AutoOffer" в продакшене.

## Обзор

Проект развертывается с использованием **Docker** и **Docker Compose**, что обеспечивает простоту деплоя и консистентность окружений.

## Варианты деплоя

1. **Docker Compose** (рекомендуется) — для большинства случаев
2. **Kubernetes** — для масштабируемых решений (опционально)

## Быстрый деплой

1. Подготовьте сервер (см. [Требования](prerequisites.md))
2. Настройте окружение (см. [Настройка окружения](environment-setup.md))
3. Разверните через Docker (см. [Деплой через Docker](docker-deployment.md))
4. Настройте SSL (см. [SSL сертификаты](ssl-certificates.md))
5. Настройте мониторинг (см. [Мониторинг](monitoring.md))

## Важные моменты

- ⚠️ **Безопасность:** Всегда используйте HTTPS в продакшене
- ⚠️ **Секреты:** Никогда не коммитьте секреты в репозиторий
- ⚠️ **Бэкапы:** Настройте регулярное резервное копирование БД
- ⚠️ **Мониторинг:** Настройте мониторинг и алерты
