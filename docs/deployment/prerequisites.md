# Требования к серверу

Требования к инфраструктуре для развертывания проекта "AutoOffer".

## Минимальные требования

### Аппаратные требования

- **CPU:** 2 ядра
- **RAM:** 4 GB
- **Диск:** 20 GB свободного места (SSD рекомендуется)
- **Сеть:** Стабильное интернет-соединение

### Программное обеспечение

- **ОС:** Linux (Ubuntu 20.04+ или Debian 11+), macOS, или Windows Server
- **Docker:** 20.10+
- **Docker Compose:** 2.0+
- **Git:** для клонирования репозитория

## Рекомендуемые требования

### Аппаратные требования

- **CPU:** 4+ ядра
- **RAM:** 8+ GB
- **Диск:** 50+ GB свободного места на SSD
- **Сеть:** Высокоскоростное соединение

### Программное обеспечение

- **ОС:** Ubuntu 22.04 LTS или Debian 12
- **Docker:** последняя стабильная версия
- **Nginx:** для reverse proxy (опционально, но рекомендуется)

## Требования для продакшена

### Аппаратные требования

- **CPU:** 4-8 ядер
- **RAM:** 16+ GB
- **Диск:** 100+ GB на SSD с резервом для логов и бэкапов
- **Сеть:** Выделенный IP, стабильное соединение

### Дополнительные требования

- **SSL сертификат:** для HTTPS (Let's Encrypt рекомендуется)
- **Доменное имя:** настроенное DNS
- **Мониторинг:** система мониторинга (опционально)
- **Бэкапы:** автоматическое резервное копирование БД

## Проверка готовности сервера

### Проверка Docker

```bash
docker --version
docker compose version
```

### Проверка дискового пространства

```bash
df -h
```

### Проверка памяти

```bash
free -h
```

### Проверка сети

```bash
ping -c 4 8.8.8.8
curl -I https://api.hh.ru
```
