# Поиск и фильтрация вакансий

Руководство по поиску и фильтрации вакансий в системе "Вкатился".

## Поиск вакансий

### Для конкретного резюме

1. Откройте резюме
2. Перейдите на вкладку "Вакансии"
3. Нажмите "Поиск вакансий"
4. Система найдет релевантные вакансии на основе:
   - Текста резюме
   - Настроек фильтров
   - Оценки AI агента

## Фильтрация вакансий

Вакансии автоматически фильтруются по:

- **Релевантности** — соответствие резюме и вакансии
- **Зарплате** — минимальная зарплата из настроек
- **Региону** — регион из настроек
- **Ключевым словам** — наличие обязательных слов
- **Стоп-словам** — отсутствие исключаемых слов
- **Компаниям** — исключение определенных компаний

## Просмотр деталей вакансии

1. В списке вакансий нажмите на вакансию
2. Откроется детальная информация:
   - Описание вакансии
   - Требования
   - Условия работы
   - Оценка релевантности

## Оценка релевантности

Каждая вакансия получает оценку релевантности (0-100):

- **80-100** — очень релевантная вакансия
- **60-79** — релевантная вакансия
- **40-59** — частично релевантная
- **0-39** — низкая релевантность

## Сохранение вакансий

Вакансии автоматически сохраняются при поиске. Вы можете:

- Просматривать сохраненные вакансии
- Отслеживать статус откликов
- Просматривать историю взаимодействий

## Генерация сопроводительного письма

Для каждой вакансии можно сгенерировать сопроводительное письмо:

1. Откройте вакансию
2. Нажмите "Сгенерировать письмо"
3. AI создаст уникальное письмо на основе:
   - Вашего резюме
   - Описания вакансии
   - Требований работодателя

## Отклик на вакансию

### Вручную

1. Откройте вакансию
2. Сгенерируйте или введите сопроводительное письмо
3. Нажмите "Откликнуться"

### Автоматически

Настройте [автоотклики](auto-replies.md) для автоматической отправки откликов.
