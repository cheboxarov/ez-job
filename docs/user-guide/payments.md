# Оплата подписки

Руководство по оплате подписки в системе "Вкатился".

## Процесс оплаты

### 1. Выбор плана

1. Перейдите в раздел "Планы"
2. Выберите подходящий план
3. Нажмите "Выбрать план"

### 2. Создание платежа

Система создаст платеж и перенаправит вас на страницу оплаты ЮКассы.

### 3. Оплата

На странице ЮКассы вы можете оплатить:

- Банковской картой
- Через СБП (Система быстрых платежей)
- Электронными кошельками
- Другими способами, доступными в ЮКассе

### 4. Подтверждение

После успешной оплаты:

1. Вы будете перенаправлены обратно в систему
2. Подписка активируется автоматически
3. Вы получите уведомление об успешной оплате

## Проверка статуса платежа

### Через веб-интерфейс

1. Перейдите в раздел "Подписка"
2. Просмотрите статус последнего платежа

### Через API

```bash
curl -X GET "http://localhost:8000/api/payments/{payment_id}" \
  -H "Authorization: Bearer $TOKEN"
```

## Способы оплаты

ЮКасса поддерживает множество способов оплаты:

- **Банковские карты** — Visa, MasterCard, МИР
- **СБП** — Система быстрых платежей
- **Электронные кошельки** — Яндекс.Деньги, Qiwi и др.
- **Банковские переводы**

## Возвраты

Возврат средств возможен только через администратора.

Для запроса возврата:

1. Свяжитесь с поддержкой
2. Укажите причину возврата
3. Администратор обработает запрос

## Чеки

Чеки об оплате автоматически отправляются на ваш email адрес, указанный при регистрации.

## Проблемы с оплатой?

Если у вас возникли проблемы:

1. Проверьте статус платежа
2. Убедитесь, что платеж не был отклонен банком
3. Свяжитесь с поддержкой
