# Интеграция с Telegram

Руководство по настройке и использованию Telegram интеграции.

## Привязка Telegram аккаунта

### Через веб-интерфейс

1. Перейдите в раздел "Настройки" → "Telegram"
2. Нажмите "Привязать Telegram"
3. Вы получите ссылку для привязки
4. Откройте ссылку в Telegram
5. Нажмите "Start" в боте
6. Аккаунт будет привязан

### Процесс привязки

1. Система генерирует временную ссылку
2. Вы открываете ссылку: `https://t.me/your_bot?start=token`
3. Бот получает команду `/start token`
4. Бот привязывает ваш Telegram аккаунт к системе
5. Вы получаете подтверждение

## Настройка уведомлений

После привязки аккаунта настройте уведомления:

1. Перейдите в "Настройки" → "Telegram"
2. Включите нужные типы уведомлений:
   - **Новые сообщения** — уведомления о новых сообщениях от работодателей
   - **Приглашения** — уведомления о приглашениях на собеседование
   - **Отказы** — уведомления об отказах (опционально)
3. Сохраните настройки

## Использование бота

### Команды бота

- `/start` — начало работы с ботом
- `/help` — справка по командам
- `/status` — статус привязки

### Inline кнопки

В уведомлениях бот предоставляет inline кнопки:

- **Просмотреть вакансию** — переход к вакансии
- **Открыть чат** — переход к чату с работодателем
- **Просмотреть отклик** — просмотр деталей отклика

## Отвязка аккаунта

Если вы хотите отвязать Telegram аккаунт:

1. Перейдите в "Настройки" → "Telegram"
2. Нажмите "Отвязать Telegram"
3. Подтвердите отвязку

После отвязки уведомления в Telegram прекратятся.

## Безопасность

- Ссылка для привязки действительна ограниченное время
- Только вы можете привязать свой Telegram аккаунт
- Уведомления отправляются только вам
