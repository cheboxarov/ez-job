# Работа с резюме

Руководство по работе с резюме в системе "AutoOffer".

## Создание резюме

### Через веб-интерфейс

1. Войдите в систему
2. Перейдите в раздел "Резюме"
3. Нажмите "Создать резюме"
4. Введите текст резюме
5. Добавьте дополнительные параметры (опционально)
6. Нажмите "Сохранить"

### Импорт из HeadHunter

1. Убедитесь, что вы авторизованы через HeadHunter
2. В разделе "Резюме" нажмите "Импортировать из HeadHunter"
3. Выберите резюме для импорта
4. Резюме будет автоматически импортировано

## Редактирование резюме

1. Откройте резюме
2. Нажмите "Редактировать"
3. Внесите изменения
4. Нажмите "Сохранить"

## Удаление резюме

1. Откройте резюме
2. Нажмите "Удалить"
3. Подтвердите удаление

**Внимание:** Удаление резюме необратимо!

## Настройка фильтров

Для каждого резюме можно настроить фильтры поиска вакансий:

1. Откройте резюме
2. Перейдите на вкладку "Настройки фильтров"
3. Настройте параметры:
   - **Минимальная зарплата** — минимальная желаемая зарплата
   - **Регион** — регион поиска
   - **Ключевые слова** — обязательные слова в вакансии
   - **Стоп-слова** — слова, при наличии которых вакансия будет исключена
   - **Исключаемые компании** — компании, вакансии которых нужно исключить
4. Нажмите "Сохранить"

## Настройка автооткликов

1. Откройте резюме
2. Включите опцию "Автоотклики"
3. Установите порог совпадения (autolike_threshold):
   - 0-50 — низкий порог (больше откликов)
   - 50-70 — средний порог
   - 70-100 — высокий порог (только очень релевантные вакансии)
