# Настройка автооткликов

Руководство по настройке и использованию автоматических откликов.

## Что такое автоотклики

Автоотклики — это автоматическая отправка откликов на релевантные вакансии без вашего участия. Система работает 24/7 и отправляет отклики сразу после публикации вакансии.

## Настройка автооткликов

### Включение автооткликов

1. Откройте резюме
2. Включите опцию "Автоотклики"
3. Установите порог совпадения (autolike_threshold):
   - **0-50** — низкий порог (больше откликов, но менее релевантные)
   - **50-70** — средний порог (рекомендуется)
   - **70-100** — высокий порог (только очень релевантные вакансии)

### Настройка фильтров

Перед включением автооткликов настройте фильтры:

1. Минимальная зарплата
2. Регион поиска
3. Ключевые слова
4. Стоп-слова
5. Исключаемые компании

## Генерация сопроводительных писем

Система автоматически генерирует уникальные сопроводительные письма для каждой вакансии на основе:

- Вашего резюме
- Описания вакансии
- Требований работодателя

Письма создаются с помощью AI и подчеркивают именно те навыки, которые нужны работодателю.

## Мониторинг откликов

### Просмотр отправленных откликов

1. Перейдите в раздел "Отклики"
2. Просмотрите список всех отправленных откликов
3. Отслеживайте статус каждого отклика

### Статистика

В разделе "Статистика" вы можете увидеть:

- Количество отправленных откликов
- Количество просмотров вакансий
- Количество приглашений
- Процент откликов с ответом

## Управление автооткликами

### Временная остановка

1. Откройте резюме
2. Выключите опцию "Автоотклики"
3. Автоотклики будут остановлены

### Изменение порога

1. Откройте резюме
2. Измените значение порога совпадения
3. Изменения вступят в силу сразу

## Лимиты подписки

Количество автооткликов ограничено вашим планом подписки:

- **FREE** — 10 откликов в месяц
- **PLAN_1** — 50 откликов в день
- **PLAN_2** — 100 откликов в день
- **PLAN_3** — 200 откликов в день

При достижении лимита автоотклики автоматически останавливаются до сброса лимита.

## Безопасность

- Система работает через официальное API HeadHunter
- Эмулирует действия реального пользователя
- Соблюдает все ограничения HeadHunter
- Не нарушает правила использования HeadHunter
