# Руководство пользователя

Добро пожаловать в руководство пользователя проекта "AutoOffer"!

## О руководстве

Это руководство предназначено для конечных пользователей сервиса и поможет вам максимально эффективно использовать все возможности системы.

## Структура руководства

- **[Регистрация и вход](registration.md)** — создание аккаунта и вход в систему
- **[Работа с резюме](resumes.md)** — создание, редактирование и управление резюме
- **[Поиск вакансий](vacancies.md)** — поиск и фильтрация вакансий
- **[Автоотклики](auto-replies.md)** — настройка автоматических откликов
- **[Управление подпиской](subscriptions.md)** — выбор плана и управление подпиской
- **[Оплата подписки](payments.md)** — процесс оплаты и управление платежами
- **[Интеграция с Telegram](telegram.md)** — настройка Telegram уведомлений
- **[FAQ](faq.md)** — часто задаваемые вопросы

## Быстрый старт

1. [Зарегистрируйтесь](registration.md) или войдите в систему
2. [Создайте или импортируйте резюме](resumes.md)
3. [Настройте фильтры](vacancies.md) для поиска вакансий
4. [Выберите план подписки](subscriptions.md) и [оплатите](payments.md)
5. [Настройте автоотклики](auto-replies.md)
6. [Подключите Telegram](telegram.md) для уведомлений

## Основные возможности

- ✅ **AI-фильтрация вакансий** — умный поиск релевантных вакансий
- ✅ **Генерация писем** — автоматическое создание сопроводительных писем
- ✅ **Автоотклики** — автоматическая отправка откликов 24/7
- ✅ **Telegram уведомления** — получение уведомлений о новых сообщениях
- ✅ **Аналитика** — отслеживание статистики откликов
