# Регистрация и вход

Руководство по регистрации и входу в систему "AutoOffer".

## Регистрация аккаунта

### Через веб-интерфейс

1. Откройте главную страницу
2. Нажмите кнопку "Регистрация"
3. Заполните форму:
   - Email адрес
   - Пароль (минимум 8 символов)
4. Нажмите "Зарегистрироваться"
5. Проверьте email для верификации (опционально)

### Через API

```bash
curl -X POST http://localhost:8000/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "securepassword123"
  }'
```

## Вход в систему

### Через веб-интерфейс

1. Откройте страницу входа
2. Введите email и пароль
3. Нажмите "Войти"

### Через API

```bash
curl -X POST http://localhost:8000/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "user@example.com",
    "password": "securepassword123"
  }'
```

Ответ содержит `access_token` для использования в API.

## Восстановление пароля

### Запрос восстановления

1. На странице входа нажмите "Забыли пароль?"
2. Введите ваш email
3. Проверьте почту для получения ссылки восстановления

### Сброс пароля

1. Перейдите по ссылке из email
2. Введите новый пароль
3. Подтвердите новый пароль

## Вход через HeadHunter

Альтернативный способ входа через аккаунт HeadHunter.

1. На странице входа выберите "Войти через HeadHunter"
2. Вы будете перенаправлены на HeadHunter
3. Авторизуйтесь в HeadHunter
4. После успешной авторизации вы вернетесь в систему

**Преимущества:**
- Не нужно создавать отдельный аккаунт
- Автоматическая синхронизация с HeadHunter
