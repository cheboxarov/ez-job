# Компоненты системы

Обзор компонентов проекта "AutoOffer" и их взаимодействие.

## Обзор

Проект состоит из нескольких ключевых компонентов, каждый из которых выполняет свою роль в общей системе.

## Компоненты

- **[Воркеры](workers.md)** — фоновые процессы для обработки задач
- **[AI агенты](agents.md)** — компоненты для работы с LLM
- **[Интеграция с HeadHunter](hh-integration.md)** — работа с API HeadHunter
- **[WebSocket](websocket.md)** — real-time коммуникация
- **[Telegram бот](telegram-bot.md)** — Telegram интеграция

## Взаимодействие компонентов

1. **Frontend** взаимодействует с **Backend** через REST API и WebSocket
2. **Workers** обрабатывают фоновые задачи (отклики, анализ чатов)
3. **Agents** используют LLM для фильтрации и генерации текстов
4. **HH Client** взаимодействует с API HeadHunter
5. **Telegram Bot** отправляет уведомления пользователям
