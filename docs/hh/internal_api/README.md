## Внутреннее фронтовое API hh.ru

### Структура раздела

- **`search-vacancy.md`** — поиск вакансий для соискателя (`GET /search/vacancy`), внутренняя XHR‑ручка, используемая на странице поиска.

### Общие замечания

- **Назначение раздела** — зафиксировать реальные фронтовые запросы hh.ru (URL, метод, заголовки, куки, query‑параметры и форму ответа), чтобы было проще:
  - воспроизводить их в автотестах / скриптах;
  - эмулировать поведение фронта в сторонних сервисах;
  - отслеживать изменения контрактов при обновлениях фронта.
- **Статус**: документация основана на «живом» трафике и может расходиться с официальным публичным API, контракт не гарантирован и может меняться без уведомления.
