# Этап 1: Миграция БД

## Описание
Создание миграции Alembic для добавления поля `autolike_threshold` в таблицу `resumes` с дефолтным значением 50% и обновлением существующих записей.

## Подзадачи

1. **Создание миграции Alembic**
   - Технологии: Alembic, SQLAlchemy
   - Добавить колонку `autolike_threshold` типа INTEGER
   - Установить NOT NULL constraint
   - Установить DEFAULT 50
   - Добавить CHECK constraint для валидации диапазона (0-100)
   - Обновить все существующие записи: SET autolike_threshold = 50

2. **Проверка миграции**
   - Выполнить `alembic upgrade head`
   - Проверить создание колонки в БД
   - Проверить дефолтное значение для новых записей
   - Проверить, что все существующие записи имеют значение 50

## Зависимости
- Нет зависимостей (первый этап)

## Критерии завершения
- [ ] Миграция создана и применена
- [ ] Колонка `autolike_threshold` существует в таблице `resumes`
- [ ] DEFAULT значение установлено на 50
- [ ] CHECK constraint работает (валидация 0-100)
- [ ] Все существующие записи обновлены значением 50
