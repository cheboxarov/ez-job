# Этап 6: Frontend

## Описание
Обновление TypeScript типов и добавление слайдера для настройки порога автолика на странице детального просмотра резюме.

## Подзадачи

1. **Обновление TypeScript типов**
   - Технологии: TypeScript
   - Файл: `frontend/src/types/api.ts`
   - Добавить поле `autolike_threshold: number` в интерфейс `Resume`
   - Добавить поле `autolike_threshold?: number | null` в интерфейс `UpdateResumeRequest`

2. **Добавление слайдера на странице ResumeDetailPage**
   - Технологии: React, TypeScript, Ant Design
   - Файл: `frontend/src/pages/ResumeDetailPage.tsx`
   - Импортировать компонент `Slider` из `antd`
   - Добавить состояние для отслеживания изменений порога (если нужно)
   - В плашке "Автоотклик" (после Switch, внутри блока `{resume?.is_auto_reply && ...}`) добавить:
     - Заголовок/описание для слайдера (например, "Порог автолика")
     - Компонент Slider с настройками:
       - min: 0
       - max: 100
       - value: resume?.autolike_threshold || 50
       - onChange: обработчик для обновления значения
       - marks: опционально, для отображения меток (0, 50, 100)
       - tooltip: отображение текущего значения в процентах
   - Добавить обработчик изменения слайдера:
     - При изменении значения вызывать `updateResume` с новым `autolike_threshold`
     - Показывать loading состояние во время сохранения
     - Показывать success/error сообщения
   - Добавить стилизацию для слайдера (соответствующую дизайну плашки)

3. **Обработка дефолтного значения**
   - Если `resume.autolike_threshold` отсутствует или null, использовать 50 как дефолт
   - Убедиться, что слайдер корректно отображает значение при загрузке страницы
